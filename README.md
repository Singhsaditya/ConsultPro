ConsultPro

ConsultPro is a full-stack web application built for a business consulting firm.
It has a public landing page for users and an admin panel to manage projects, clients, contacts, and newsletter subscribers.

This project was built as part of a placement process to demonstrate full-stack development skills.

Features

Business consulting landing page

Admin dashboard

Manage projects from admin panel

Manage client testimonials

View contact form submissions

View newsletter subscribers

Fully responsive UI

Tech Stack

Frontend

React (Vite)

Tailwind CSS

React Router

Backend

Node.js

Express.js

MongoDB Atlas

Mongoose

Project Structure
ConsultPro/
├── frontend/
├── backend/
└── README.md

How It Works

Users can view projects and clients on the landing page

Users can submit contact forms and subscribe to the newsletter

Admin can manage all data from the admin panel

All data is stored in MongoDB

Run Locally
Backend
cd backend
npm install
node server.js

Frontend
cd frontend
npm install
npm run dev

Deployment

Backend deployed on Render

Frontend deployed on Vercel

MongoDB Atlas used as database
